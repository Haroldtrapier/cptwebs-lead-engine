# CPTWEBS Lead Engine

CPTWEBS Lead Engine is a Vercel-ready Next.js MVP for finding, scoring, and prioritizing churches and small businesses that need modern websites.

## Features

- Dashboard metrics for lead pipeline and revenue estimates
- Lead table with filters and priority scoring
- Lead search form using mock data
- Outreach generator for cold email, SMS, phone script, follow-up, and website audit summaries
- CSV export for visible leads

## Local setup

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
```

## Deploy to Vercel

Import this GitHub repository into Vercel as a Next.js project and deploy with the default settings.

## Compliance notes

Use public business information only. Do not scrape private personal information. Respect platform terms and robots.txt. Position outreach as a personalized website audit and free homepage mockup, not spam automation.
